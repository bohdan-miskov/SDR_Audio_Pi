# Audio Drone Radar — Система Акустичного Виявлення 🛰️

**Audio Drone Radar** — це інтелектуальна система моніторингу повітряного простору, розроблена на Python. Програма аналізує звуковий фон у реальному часі, виявляє специфічні акустичні підписи FPV-дронів за допомогою математичних методів та нейромереж.

> **Архітектура:** Проект складається з графічного інтерфейсу (UI), модуля цифрової обробки сигналів (DSP) та інтегрованої згорткової нейромережі (CNN).

## 🎯 Основні можливості

* **Детекція через ШІ** — класифікація звуку за допомогою згорткової нейромережі (CNN) для мінімізації помилкових спрацювань.
* **Аналіз гармонік (Goertzel)** — математичний пошук частот моторів дронів із динамічною підсвіткою на графіку.
* **Живий спектр** — візуалізація аудіоефіру в реальному часі з логарифмічним масштабом (дБ).
* **Smart Logger** — автоматичний запис інцидентів у форматі **FLAC** (стиснення без втрат) із системою ротації та очищення старих файлів.
* **Triangulation Ready** — вбудований мережевий емітер для передачі даних на центральний сервер (TDoA).

## 📋 Вимоги

* **ОС:** Windows 10/11, Linux (рекомендовано Raspberry Pi OS).
* **Python:** 3.12 (рекомендовано для стабільної роботи TensorFlow).
* **Залізо:** USB-мікрофон або спрямована мікрофонна решітка.

## ⚙️ Встановлення та Налаштування

### 1. Створення віртуального оточення
Рекомендовано використовувати ізольоване середовище `.venv`:

```bash
# Створення
python -m venv .venv

# Активація (Windows):
source ./.venv/Scripts/activate

# Активація (Linux/macOS):
source .venv/bin/activate


pip install -r requirements.txt


3. Налаштування конфігурації
У корені проекту знайдіть файл config.yaml. Відкрийте його для налаштування:

goertzel_frequencies — цільові частоти моторів.

threshold_confidence — поріг впевненості нейромережі.

max_storage_mb — ліміт місця під аудіозаписи.

network — налаштування IP та портів для тріангуляції.

Запуск
Основний режим (GUI)
